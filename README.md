## 1. 核心哲學命題（v0.3 確認版）

6. 【v0.2】身心不是二元對立，而是同一莫比烏斯曲面的兩個觀察角度──進化是讓這個環持續膨脹與螺旋上升

## 5. 知行合一測量框架

> 這是 v0.3 最重要的實踐突破，回應了 Q-10（L0 操作性測量）這個跨版本最高優先問題。王陽明「知行合一」不只是哲學──它是 L0 連通度的操作性觀測協議。

### 5.2 ZHI 測量協議

**第一層：意念-響應延遲（τ_lag）**
- 設備：消費級 EEG（Muse/Emotiv）+ HRV 穿戴（Apple Watch/Polar H10）
- 方法：受試者在接收到聲音指令後，以意念（而非主動動作）引導心率或呼吸變化
- 測量：EEG 運動意圖前兆出現時間 → HRV 響應出現時間的時間差 = τ_lag
- 指標意義：τ_lag 縮短 → L0 通道更暢通 → κ_L0 提升

**第二層：信號保真度（η_faith）**
- η_faith = Corr(EEG_γ, ΔHR)，值域 [0, 1]
- 指標意義：η_faith = 1 代表意念強度完全忠實轉化為生理響應

**第三層：I_forced 驗證**
- Readiness Potential 檢測：確認行為由內部意圖發起
- 此層確保 E-23 的前提條件（I_forced ≡ 0）成立，數據方為有效
